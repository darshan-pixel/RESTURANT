# 🍽️ Restaurant Menu Webpage (HTML Only)

## 📌 Project Overview
This project focuses on building a simple Restaurant Menu Webpage using only HTML.  
Students will learn how to structure a webpage and display content like menus, images, and tables.

---

## 🎯 Objective
- Understand basic HTML structure
- Practice using different HTML tags
- Create a real-world static webpage

---

## 🛠️ Technologies Used
- HTML5 (Only)

---

## 📂 Project Requirements

### 1. Basic Structure
- Use proper HTML5 boilerplate:
  - <!DOCTYPE html>
  - <html>, <head>, <body>

---

### 2. Sections to Include

#### 🍴 Header
- Restaurant Name (h1)
- Tagline (p)

#### 📖 Menu Categories
- At least 3 categories:
  - Starters
  - Main Course
  - Desserts

#### 🍕 Menu Items
- Use lists (ul/ol)
- Each item must include:
  - Dish Name
  - Description
  - Price

#### 🖼️ Images Section
- Add at least 2 food images

#### 🧾 Price Table
- Table should include:
  - Dish Name
  - Category
  - Price

#### 📞 Contact Section
- Address
- Phone Number
- Email (mailto link)

---

## 📚 HTML Concepts Covered
- Headings (h1 - h3)
- Paragraphs (p)
- Lists (ul, ol)
- Tables (table, tr, td)
- Images (img)
- Links (a)

---

## 🚫 Restrictions
- ❌ No CSS
- ❌ No JavaScript
- ❌ No frameworks

---

## ⭐ Bonus Features (Optional)
- Internal navigation links
- Use of <hr> for section separation
- “Today’s Special” section

---

## 🧑‍🏫 Evaluation Criteria
- Proper HTML structure
- Correct use of tags
- Completeness of sections
- Clean and readable code
- Creativity

---

## 📁 Submission Guidelines
- Submit a single `.html` file
- File name: `restaurant-menu.html`

---

## 🚀 Expected Outcome
A well-structured static webpage displaying a restaurant menu using only HTML.
